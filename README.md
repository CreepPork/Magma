# Usage
<!-- usage -->
```sh-session
$ npm install -g magma
$ magma COMMAND
running command...
$ magma (-v|--version|version)
magma/2.0.0 linux-x64 node-v10.16.0
$ magma --help [COMMAND]
USAGE
  $ magma COMMAND
...
```
